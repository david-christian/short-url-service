## Synopsis

### nodejs Express
專案構成語言及語言框架
### MYSQL
RDBMS 關聯式資料庫
### REDIS
NOSQL 做熱資料快取
### crypto-js/md5
雜湊套件，短網址隨機ID使用 md5 生成
### open-graph-scraper
用於從網站上抓取 Open Graph
### SQL Prepared Statement
參數化查詢，防禦SQL Injection攻擊。效能上佔有優勢。
